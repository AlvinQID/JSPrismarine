# Running JSPrismarine

## 1. Prerequisites

-   1.1 Node 14+ & NPM 7.
-   1.2 lerna (Lerna is our monorepo manager of choice which can be installed with a simple `$ npm install -g lerna`).

## Install dependencies

```bash
    # Install npm dependencies
    $ npm install

    # Setup monorepo
    $ lerna bootstrap
```
